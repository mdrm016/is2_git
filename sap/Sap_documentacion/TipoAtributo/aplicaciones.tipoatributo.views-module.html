administrarTipoAtributo(request, id_proyecto)

source code 

Recibe un request, obtiene la lista de todos los Tipos de Atributo de un Proyecto y luego retorna el html renderizado con la lista de Tipos de atributo

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la solic. web actual que llamo a esta vista
  • id_proyecto (Integer) - Es el id del proyecto actual en el que se esta trabajando
Returns: django.http.HttpResponse
tipo_atributos.html, donde se listan los Tipos de atributos, ademas de las funcionalidades para un Tipo de Atributo

eliminarTipoAtributo(request, id_proyecto, id_tipo_atributo)

source code 

Eliminar de manera logica los registros del Tipo de atributo.Tambien elimina la relacion entre los proyectos que poseen este Tipo de atributo.

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la solicitud web actual que llamo a esta vista
  • id_proyecto (Integer) - Es el id del proyecto actual en el que se esta trabajando
  • id_tipo_atributo (Integer) - Es el id del Tipo de Atributo a eliminar
Returns: django.shortcuts.render_to_response
Se retorna al la administracion de Roles o se manda a la pagina de notificacion
Decorators:
