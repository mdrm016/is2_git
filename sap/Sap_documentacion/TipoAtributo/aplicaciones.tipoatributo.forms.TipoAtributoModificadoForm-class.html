                   TipoAtributoModificadoForm

Campos del Tipo de atributo necesarios para el registro en la base de datos de los datos modificados de un Tipo de atributo. Este formulario con los campos descritos son enviados al template html encargado de tomar los datos de registro. Control de datos ingresados por el usuario.
