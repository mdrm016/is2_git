adm_proyectos(request)

source code 

Recibe un request, se verifica cual es el usuario registrado y se obtiene la lista de proyectos con los que esta relacionado desplegandolo en pantalla, ademas permite realizar busquedas avanzadas sobre los los proyectos que puede mostrar. Si el usuario es el administrador despliega todos los proyectos.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista adm_proyectos.
Returns: django.shortcuts.render_to_response.
index.html, donde se listan los proyectos, ademas de las funcionalidades para cada proyecto.
Decorators:
  • @login_required(login_url= '/login/')

Author: Marcelo Denis.
