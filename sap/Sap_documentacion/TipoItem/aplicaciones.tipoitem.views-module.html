adm_tipoitem(request, id_proyecto)

source code 

Recibe un request y un id de proyecto, se obtiene todos los tipos de item disponibles para ese proyecto y luego son enviados al template para desplegarlos al usuario solicitante, ademas de realizar busquedas avanzadas de los tipos de items desplegados.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista adm_tipoitem.
  • id_proyecto (string.) - Contiene el id del proyecto cuyos tipos de items seran desplegados.
Returns: django.shortcuts.render_to_response.
tipoitem.html, donde se listan los tipos de items, ademas de las funcionalidades para cada tipo de item.
Decorators:
  • @login_required(login_url= '/login/')
  • @permission_required('tipoitem.administrar_tipoitem', raise_exception= True)

  • proyecto_select (string.) - Contiene el id del proyecto al cual pertenece el tipo de item que sera importado.
  • id_tipoitem (string.) - Contiene el id del tipo de item a importar.
Returns: django.shortcuts.render_to_response.
creartipoitemimportado.html, donde se redirige al usuario para completar los datos del nuevo tipo de item a importar y luego a proyectoalerta.html donde se notifica que el proyecto fue importado correctamente.
