                        TipoItem

El modelo TipoItem describe la estructura de cada instancia de un Tipo de Item, los campos que contiene el modelo son: nombre: campo de tipo texto que contendra el nombre del Tipo de Item. descripcion: campo de tipo texto que contendra una breve descripcion del Tipo de Item. id_proyecto: campo que contendra el id del proyecto al cual pertenece el tipo de item. is_active: campo de tipo logico que indicara si el Tipo de Item esta eliminado. listaAtributo: campo que listara la serie de atributos de los que esta compuesto el Tipo de Item. Los Tipos de Item seran ordenados en la tabla por nombre.
