Class UsuarioModificadoForm

source code

django.forms.Form --+
                    |
                   UsuarioModificadoForm

Atributos de Usuario necesarios para el registro en la base de datos de un Usuario a modificar. Este formulario con los campos descritos son enviados al template html encargado de desplegar los datos del usuario a modificar. Control de datos ingresados por el usuario.
