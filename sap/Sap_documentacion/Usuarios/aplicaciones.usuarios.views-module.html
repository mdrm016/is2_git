modificarUsuario(request, id_usuario)

source code 

Busca en la base de datos al usuario cuyos datos se quieren modificar. Presenta esos datos en un formulario y luego se guardan los cambios realizados. Con la posibilidad de que el usuario cancele la operacion

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la solic. web actual que llamo a esta vista
  • id_usuario (integer) - es el id del usuario cuyos datos se quieren modificar
Returns: django.HttpResponse
modificar_usuario.html,un formulario donde se despliegan los datos que el usuario puede modificar ,usuario_modificado.html, donde se notifica al usuario el exito de la operacion
Decorators:
  • @login_required(login_url= '/login/')

Author: eduardo gimenez

usuario_eliminar(request, id_usuario)

source code 

Comprueba que el id del usuario a eliminar no se sea del administrador, caso contrario procede a eliminar de la base de datos los registros del usuario.

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la solicitud web actual que llamo a esta vista
  • id_usuario (string) - Contiene el id del usuario a ser eliminado.
Returns: django.shortcuts.render_to_response
Se retorna al la administracion de usuarios o se manda a la pagina de notificacion
Decorators:
  • @login_required(login_url= '/login/')
