adm_relaciones(request, id_proyecto, id_fase, id_item)

source code 

Recibe un request, se verifica cual es el usuario registrado y el proyecto del cual se solicita, se obtiene la lista de fases con las que estan relacionados el usuario y el proyecto desplegandola en pantalla, ademas permite realizar busquedas avanzadas sobre las fases que puede mostrar.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista.
Returns: django.shortcuts.render_to_response.
fases.html, donde se listan las fases, ademas de las funcionalidades para cada fase.
