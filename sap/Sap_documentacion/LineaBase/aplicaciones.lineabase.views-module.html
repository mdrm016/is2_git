source code 

Recibe un request, se obtiene la lista de Lineas Base con las que estan relacionados el proyecto y la fase desplegandola en pantalla, ademas permite realizar busquedas avanzadas sobre las fases que puede mostrar.

  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista.
Returns: django.http.HttpResponse
lineas_base.html, donde se listan las Lineas Base, ademas de las funcionalidades para cada Linea Base.

Author: Eduardo Gimenez.

generarLineaBase(request, id_proyecto, id_fase)

source code 

Recibe un request, se verifica cual es el usuario registrado, el proyecto del cual se solicita, la fase en la que se esta trabajando, de acuerdo a si el rol del usuario esta o no relacionado con el proyecto y la fase actual, se despliega un formulario para Generar la Linea base, en donde se puede agregar una descripcion de la Linea Base y tambien los items que se pueden incluir en la Linea Base. Una vez que el usuario levanta el formulario hacia el servidor, se realizado los controles y las operaciones para poder generar la linea base de manera correcta.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista.
  • id_proyecto (Integer) - Es el id del proyecto actual en el que se esta trabajando
  • id_fase (Integer) - Es el id de la fase actual en el que se esta trabajando
Returns: django.http.HttpResponse
alerta_linea_base.html, mensaje de exito
Decorators:
  • @login_required(login_url= '/login/')
  • @permission_required('lineabase.generar_linea_base', raise_exception= True)

Author: Eduardo Gimenez.

consultar_lineabase(request, id_proyecto, id_fase, id_lineabase)

source code 

Recibe un request, el id de proyecto, el id de fase y el id de la linea base a ser consultada, se verifica si el usuario tiene permisos para consultar los datos de una linea base y se lo redirige a una pagina que despliega los datos de la linea base solicitada.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista consultar_tipoitem.
  • id_proyecto (string.) - Contiene el id del proyecto al cual pertenece el tipo de item a consultar.
  • id_fase (string.) - Contiene el id de la fase a la cual pertenece la linea base a consultar.
  • id_lineabase (string.) - Contiene el id de la linea base a consultar.
Returns: django.shortcuts.render_to_response.
consultarlineabase.html, donde se encuentra la pagina de consulta de linea base.
Decorators:
  • @login_required(login_url= '/login/')
  • @permission_required('lineabase.consultar_linea_base', raise_exception= True)

Author: Marcelo Denis.

informe_lineabase(request, id_proyecto, id_fase, id_lineabase)

source code 

Recibe un request, el id de proyecto, el id de fase y el id de la linea base de la cual se generara el informe, se verifica si el usuario tiene permisos para empezar el procedimiento de generacion de informe de una linea base y se lo redirige a un archivo pdf que contirne los datos de la linea base solicitada.

Parameters:
  • request (django.http.HttpRequest.) - Contiene informacion sobre la solicitud web actual que llamo a esta vista consultar_tipoitem.
  • id_proyecto (string.) - Contiene el id del proyecto al cual pertenece el tipo de item a consultar.
  • id_fase (string.) - Contiene el id de la fase a la cual pertenece la linea base a consultar.
  • id_lineabase (string.) - Contiene el id de la linea base a consultar.
Returns: PDF.
linea_base_numero.html, donde se encuentra el informe de linea base.
Decorators:
  • @login_required(login_url= '/login/')
  • @permission_required('lineabase.generar_informe_linea_base', raise_exception= True)

Author: Marcelo Denis.
