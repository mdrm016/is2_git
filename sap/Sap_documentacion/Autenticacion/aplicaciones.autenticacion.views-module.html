login_view(request)

source code 

La funcion login view tiene la logica de autenticacion, un usuario logueado en el sistema no puede moverse a la pagina del login ni ver el formulario de login, por lo que sera redirigido a la pagina de inicio si intentase acceder a la misma.

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la solicitud web actual que llamo a esta vista
Returns: django.http.HttpResponseRedirect
Se retorna al inicio o se manda a la pagina de login
